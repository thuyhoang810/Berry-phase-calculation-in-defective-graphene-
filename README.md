# Berry-phase-calculation-in-defective-graphene-
Data support for the Berry phase calculation of defective graphene:
SP: Symmetry-preserving disorder, where each sublattice contains equal numbers of the positive and negative on-site energy,
SB: Symmetry- breaking disorder, where positive and negative on-site energies are assigned exclusively to the A and B sublattices, respectively,
Pue: graphene in pristine form
